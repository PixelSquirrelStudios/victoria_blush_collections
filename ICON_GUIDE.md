# Icon Guide for Victoria Blush Collections

## Required Service Icons

Place these SVG icon files in `public/assets/icons/`:

### Icon Files Needed:

1. **scissors.svg** - Cut and Finish service
2. **sparkles.svg** - Blow Dry, Styling, and Foil services
3. **palette.svg** - Balayage and Maintenance services
4. **droplet.svg** - Global Colour and Root services
5. **waves.svg** - Toning Service
6. **help-circle.svg** - Colour Correction
7. **zap.svg** - Global Bleach
8. **star.svg** - Hair Treatments (Kerastase, K18)

## Icon Requirements:

- **Format**: SVG (vector format)
- **Size**: 32x32px viewBox recommended
- **Style**: Simple, minimalist line icons
- **Color**: Single color (monochrome) - opacity will be controlled by CSS
- **File naming**: Use kebab-case (lowercase with hyphens)

## Where to Get Icons:

### Free Icon Sources:

1. **Lucide** - https://lucide.dev

   - Modern, clean design
   - Download individual SVGs
   - Search for: scissors, sparkles, palette, droplet, waves, help-circle, zap, star

2. **Heroicons** - https://heroicons.com

   - By Tailwind CSS creators
   - Consistent style

3. **Iconmonstr** - https://iconmonstr.com

   - Large collection
   - Simple, clean icons

4. **Feather Icons** - https://feathericons.com
   - Minimalist design
   - Perfect for professional sites

### How to Download:

1. Visit the icon site
2. Search for the icon name (e.g., "scissors")
3. Download as SVG
4. Rename to match the required filename
5. Place in `public/assets/icons/`

## Example SVG Format:

Your SVG files should look something like this:

```svg
<svg xmlns="http://www.w3.org/2000/svg" width="24" height="24" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round">
  <path d="M..."/>
</svg>
```
