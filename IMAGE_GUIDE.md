# Image Guide for Victoria Blush Collections

## Required Images

### Hero Section

Place your main hero image at:

- `public/assets/images/victoria-hero.jpg`
  - Recommended size: 1200x1600px (portrait orientation)
  - High-quality professional photo of Victoria

### Gallery Section

Add 9 images to showcase your work:

- `public/assets/gallery/balayage-1.jpg`
- `public/assets/gallery/balayage-2.jpg`
- `public/assets/gallery/balayage-3.jpg`
- `public/assets/gallery/blonde-1.jpg`
- `public/assets/gallery/blonde-2.jpg`
- `public/assets/gallery/cut-1.jpg`
- `public/assets/gallery/color-1.jpg`
- `public/assets/gallery/color-2.jpg`
- `public/assets/gallery/styling-1.jpg`

Each gallery image should be:

- Recommended size: 800x1066px (3:4 aspect ratio)
- High-quality photos of your work
- Well-lit, professional shots

### Optional Pattern

For the subtle background pattern in the hero:

- `public/assets/patterns/subtle-dots.svg`
  - This is optional and can be removed from Hero.tsx if not needed

## Image Optimization Tips

1. **Format**: Use WebP or JPEG format for best performance
2. **Quality**: Balance quality vs file size (aim for under 500KB per image)
3. **Dimensions**: Resize images to the recommended dimensions before uploading
4. **Names**: Keep filenames lowercase with hyphens, no spaces

## Temporary Placeholder

Until you add real images, you can:

1. Use placeholder images from https://unsplash.com (hair/beauty themed)
2. Or temporarily use solid color placeholders
